# The `auto` keyword
The `auto` keyword has a number of different uses:

- To [[cpp.keyword.auto.decuce_func_type|deduce a function's type]]
- To [[cpp.keyword.auto.structured_binding|define structured bindings]]
- To [[cpp.keyword.auto.deduce_expr_type|deduce the type of an expression]]
- To [[cpp.keyword.auto.deduce_template_parameters|deduce the type of template parameters]]
- To [[cpp.keyword.auto.define_function_templates|define abbreviated function templates]]
- To [[cpp.keyword.auto.decltype(auto)|use with `decltype(auto)`]]
- To [[cpp.keyword.auto.alternative_function_syntax|write functions using the alternative function syntax]]
- To [[cpp.keyword.auto.generic_lamdba_expr|write generic lambda expressions]]
