C++20 and C++23 introduce `std::format`, `std::print`, `std::println` utilities to format and output string. We can contain as many [[cpp.string.format.replacement_field|replacement fields]] as we want in pattern string, then we can either specify total or none [[cpp.string.format.argument_index|argument indices]] to control how should each argument be formatted in pattern string. Furthermore, if we want to control the format in each [[cpp.string.format.replacement_field|replacement fields]], we can specify different [[cpp.string.format.format_specifiers|format specifiers]].

Besides, we can also [[cpp.string.format.format_escaped_string|format escaped characters and strings]] and [[cpp.string.format.format_ranges|format ranges]].
